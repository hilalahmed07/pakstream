Core Capabilities

A comprehensive platform designed for secure, high-performance media operations in restricted network environments.

🎬

Adaptive Video Streaming

HLS-based streaming with automatic quality adaptation (360p–1080p). FFmpeg-powered transcoding pipeline with real-time progress tracking.

📄

Document Management

Upload, process, and view PDFs with automatic page extraction, thumbnail generation, and full integrity verification.

📊

Presentation Viewer

PPT/PPTX upload with automatic conversion to slide images via LibreOffice. Built-in slide viewer with navigation.

🎥

Live Premieres

Schedule synchronized video premieres with real-time viewer synchronization via Socket.IO WebSocket connections.

🔐

Role-Based Access Control

JWT authentication with Admin/User roles. Protected routes, admin-only uploads, and secure credential management.

🛡️

File Integrity Verification

SHA-256 hash verification for all uploaded content — videos, documents, and presentations. Ensures tamper-proof delivery.

🌐

Airgapped Deployment

Full offline installation with pre-bundled node_modules, system packages, and ISO-ready deployment scripts. No internet required.

📡

Push-Based CDN

Optional MinIO S3-compatible object storage with configurable push-based CDN for multi-node content distribution.

⚡

Real-time Updates

Socket.IO integration provides live processing progress, premiere sync, and instant UI updates across all connected clients.

Deployment Options

Flexible deployment for both connected and fully airgapped environments.

🌐 Standard Deployment

  • npm install on connected machine
  • MongoDB + MinIO services via systemd
  • Nginx reverse proxy with SSL
  • PM2 or systemd process management

🔒 Airgapped Deployment

  • Pre-bundled ISO with all dependencies
  • Offline node_modules (tar.gz archives)
  • System packages for Ubuntu/Debian
  • Complete installation scripts included

📡 Push-Based CDN

  • Multi-node content distribution
  • MinIO S3-compatible object storage
  • Configurable replication policies
  • Works within airgapped networks

API Endpoints

RESTful API with rate limiting, helmet security, and comprehensive CORS support.

🔐 Authentication

  • POST /api/auth/register
  • POST /api/auth/register-admin
  • POST /api/auth/login
  • GET /api/auth/profile

🎬 Media

  • GET/POST /api/videos
  • GET/POST /api/documents
  • GET/POST /api/presentations
  • GET/POST /api/premieres

🛡️ Integrity

  • GET /api/videos/:id/hash
  • POST /api/videos/:id/verify
  • GET /api/documents/:id/hash
  • POST /api/documents/:id/verify
